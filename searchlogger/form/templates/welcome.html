Welcome to the study!

In this study, we ask you to compare the quality of support and documentation for packages. We give you two packages to compare, related to some task you might use code to do. We ask you to assess support and documentation by anticipating which package will better suit your needs if you were to use it for the programming project we give you.

For each task that we give you, we ask you to do the following: Plan how you will learn about support and documentation Use the web to learn about support and documentation Answer questions about what you learned
The second step will be timed.

When you are ready to proceed, please press the "Next" button.
